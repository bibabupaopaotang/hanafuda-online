# 菜单与大厅 — 游戏设计文档

> 校验日期：2026-04-27
> 依赖：`network.md`、`battle.md`

## 概述

- **系统名称**：菜单与大厅 (Menu & Lobby)
- **所属层级**：Presentation
- **优先级**：P3
- **设计目标**：实现游戏的主菜单、房间大厅、创建/加入房间等交互界面。

## 微信分享

```typescript
// 分享房间到微信
function shareRoom(roomId: string, mode: string, targetScore: number): void {
  wx.shareAppMessage({
    title: `来一局花札！房间号：${roomId}`,
    imageUrl: 'share_image.png',
    query: `roomId=${roomId}&mode=${mode}&target=${targetScore}`,
  });
}

// 处理分享进入
function handleShareLaunch(query: Record<string, string>): void {
  if (query.roomId) {
    joinRoom(query.roomId);
  }
}
```
